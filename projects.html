Automated Statistician

Leveraged reinforcement learning algorithms to build an automated system that performed predictive model and hyperparameter selection in a multiple-model setting. Demonstrated that the automated system achieved faster model tuning than existing model selection baselines. Completed as final project for Stanford AA228: Decision Making under Uncertainty.

Minimum Probability Flow Learning

Extended Sohl-Dickstein's work on minimum probability flow by incorporating the use of graph factorization/auxiliary Markov random fields for parameter-estimation in binary pair-wise Markov random fields. The resulting factorized minimum porbability flow is embarrassingly parallelizable and out-competes factorized maximum likelihood estimation in the parameter estimation of large Ising graphs.

Neural Net Bayesian Optimization

Implemented a distributed version of a Bayesian optimization framework that used a deep neural network as the surrogate model (based on Ryan Adams' work on scalable Bayesian optimization). This implementation was developed as part of the John Kemeny Computing Competition.

Electron Cloud Visualization

Wrote an interactive software that demonstrates the connection between the Schrodinger wavefunction and its corresponding probability density function in 3-dimensional space by visualizing the electron cloud as the set of observations drawn from the probability density functions associated with each atomic orbital.
